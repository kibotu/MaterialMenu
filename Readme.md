Small library to help getting started for creating material design navigation drawer, handling action bar and an actual usable menu. 

### How to install
	
	repositories {
	    maven {
	        url "https://jitpack.io"
	    }
	}
		
	dependencies {
            compile 'com.github.kibotu:MaterialMenu:-SNAPSHOT'
    }
